## 🚀 Proyecto destacado: ChefAI

App móvil con IA actualmente en **beta cerrada en Google Play Store**, desarrollada de forma individual end-to-end. Analiza imágenes de la nevera con GPT-4 Vision y genera recetas personalizadas según dieta, calorías objetivo y número de comensales.

**Stack:**
- 🔧 **Backend:** FastAPI + PostgreSQL (Supabase), autenticación Google OAuth 2.0 + JWT
- 📱 **Frontend:** React Native + Expo (iOS y Android)
- 🤖 **IA:** Integración con OpenAI API (GPT-4 + Vision)
- 💳 **Pagos:** Stripe (modelo freemium)
- ✅ Cumplimiento GDPR · Verificación oficial de Google OAuth · Soporte ES/EN
